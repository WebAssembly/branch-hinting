Structure¶

Code Metadata¶

A Code Metadata item is a piece of information logically attached to an instruction.

An item is associated with a format, which defines the item’s payload.

Branch Hints¶

A Branch Hint is a Code Metadata format that can be attached to br_if and if instructions. Its payload indicates whether the branch is likely or unlikely to be taken.
